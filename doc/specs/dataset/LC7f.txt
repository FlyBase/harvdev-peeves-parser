Valid format is:

Four components separated by semi-colons: file path, file type, file size and file description. Value type and value separated by colons. The value for description may be left blank.
For example:
File path: ftp://ftp.flybase.org/flybase/associated_files/MachiBase.embryo.Plus.wig ; File type: wig ; File size: 3.5 MB ; File description: plus

Similar (but not identical) in format to P38
